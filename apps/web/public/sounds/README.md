# 音效文件目录

## 文件命名规范

请将下载的音效文件放在此目录，并按照以下命名：

**电报机音效**: `telegraph-click.wav` 或 `telegraph-click.mp3`
- 从 Freesound.org 下载的电报机音效
- 文件ID: 148893 (Timbre)
- 网址: https://freesound.org/people/Timbre/sounds/148893/

## 下载步骤

### 电报机音效
1. 访问: https://freesound.org/people/Timbre/sounds/148893/
2. 登录 Freesound.org 账号（如果没有账号需要注册）
3. 点击 "Login to download" 按钮下载
4. 将文件重命名为 `telegraph-click.wav` 或 `telegraph-click.mp3`
5. 放在此目录: `apps/web/public/sounds/`

## 文件格式

- 推荐格式: **WAV** (无损，质量最好)
- 备选格式: **MP3** (文件小，兼容性好)
- 文件大小: 建议 < 100KB (短音效)

## 测试

下载文件后，在设置页面选择对应的音效模式即可测试。

如果文件不存在，系统会自动降级到代码生成的音效。
